## 🚀 우선순위별 추가 기능

### 🔥 우선순위 1 (핵심 기능 완성)

#### 1.1 콘텐츠 관리 기능
- [ ] **관리자 FAQ 관리 페이지**
  - FAQ CRUD (생성/수정/삭제)
  - 카테고리별 관리
  - 정렬 순서 조정
  - 다국어 편집

- [ ] **관리자 건강 정보(Article) 관리 페이지**
  - Article CRUD
  - 이미지 업로드
  - 태그 관리
  - 게시/비게시 설정
  - 다국어 편집

- [ ] **Q&A 상세 페이지 및 답변 기능**
  - Q&A 상세 보기
  - 관리자 답변 작성
  - 답변 수정/삭제
  - 상태 관리 (대기/답변완료/종료)

#### 1.2 검색 기능 강화
- [ ] **검색 자동완성**
  - 실시간 검색어 제안
  - 인기 검색어 표시
  - 최근 검색어 기록

- [ ] **검색 결과 개선**
  - 검색 결과 하이라이트
  - 검색 결과 통계 (총 N개 검색됨)
  - 검색 필터 조합 저장
