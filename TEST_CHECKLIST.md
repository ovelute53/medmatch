# 테스트 체크리스트

## 🎯 최근 구현 기능 테스트

### 1. 리뷰 이미지 업로드 기능

#### 1.1 리뷰 작성 시 이미지 업로드
- [ ] 이미지 선택 버튼 클릭 및 파일 선택
- [ ] 최대 5개 이미지 업로드 가능 확인
- [ ] 이미지 미리보기 표시 확인
- [ ] 이미지 개별 삭제 기능 확인
- [ ] 6개 이상 이미지 선택 시 제한 메시지 확인
- [ ] 지원하지 않는 파일 형식 업로드 시 에러 메시지 확인 (예: PDF, TXT)
- [ ] 5MB 이상 이미지 업로드 시 에러 메시지 확인

#### 1.2 이미지 업로드 및 리뷰 연결
- [ ] 리뷰 작성 시 이미지 업로드 진행 상태 표시 확인
- [ ] 리뷰 작성 완료 후 이미지가 리뷰에 연결되는지 확인
- [ ] 리뷰 상세 페이지에서 업로드한 이미지 표시 확인
- [ ] 여러 이미지의 순서가 올바르게 표시되는지 확인

#### 1.3 권한 및 보안
- [ ] 로그인하지 않은 사용자는 이미지 업로드 불가 확인
- [ ] 다른 사용자의 리뷰에 이미지 추가 불가 확인
- [ ] 관리자는 모든 리뷰에 이미지 추가 가능 확인

---

### 2. Article 관리 기능

#### 2.1 Article 작성 및 수정
- [ ] 새 Article 생성
- [ ] Article 제목(한국어/영어) 입력
- [ ] Article 본문(한국어/영어) 입력
- [ ] 요약 입력 (선택사항)
- [ ] 카테고리 선택 (일반/질병/치료/예방)
- [ ] 이미지 URL 입력
- [ ] 태그 추가 및 삭제
- [ ] 작성자 정보 입력

#### 2.2 임시저장 기능
- [ ] "게시하기" 체크박스 해제 시 "임시저장" 버튼 표시 확인
- [ ] "게시하기" 체크박스 선택 시 "게시하기" 버튼 표시 확인
- [ ] "바로 게시하기" 버튼 클릭 시 즉시 게시 확인
- [ ] 임시저장된 Article이 "임시저장" 필터에 표시되는지 확인
- [ ] 임시저장된 Article이 일반 목록에는 표시되지 않는지 확인

#### 2.3 Article 게시
- [ ] 게시 상태 필터링 (전체/게시됨/임시저장) 확인
- [ ] 게시된 Article이 프론트엔드에서 표시되는지 확인
- [ ] 게시일이 올바르게 기록되는지 확인
